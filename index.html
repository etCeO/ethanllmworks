About

Ethan LLM Works is a self-directed portfolio project that involves taking my original programs and transforming them into interactive simulations and computational tools using advanced techniques in prompt engineering. Every project starts with logic that I develop using Python, Java, or C++, and evolves into a full-fledged html web application through AI-assisted pipelines.

These projects serve as more than visual redesigns, but are a testament to my abilities in bridging conventional software development with the latest AI innovation. Although the underlying code and algorithms are mine, the web interfaces have been developed through purposeful experimentation with large language models. What I aim to achieve through this initiative is not the authorship of these websites but to showcase the application of my AI engineering skills in terms of crafting prompts and refining interfaces.

This platform is a part of an independent study into the development of LLM products, prototyping, and computational designs. It is to be taken as a way of learning about AI designs in intuitive and engaging mannerisms. Through these projects, I hope to learn more about how human-designed logic and machine-assisted generation can be combined in creating function-oriented digital systems.
